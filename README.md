🚀 TechNova S.A. — Executive Business Simulator

Sitio web del proyecto:
https://ronnyvilla3021.github.io/empresa-simulador/


🧠 Descripción

Simulación empresarial interactiva enfocada en toma de decisiones estratégicas; el usuario asume el rol de CEO de una startup SaaS; cada acción impacta métricas reales como capital, ingresos y riesgo; el sistema responde dinámicamente con análisis impulsado por inteligencia artificial; no es un CRUD, es un sistema vivo orientado a negocio.

🎯 ¿Qué demuestra este proyecto?

Pensamiento estratégico aplicado a tecnología; capacidad de modelar lógica de negocio; integración de inteligencia artificial en flujos reales; desarrollo de interfaces modernas tipo SaaS; manejo de estado dinámico sin frameworks.

⚙️ Stack Tecnológico
HTML5; estructura completa de la aplicación
CSS3; diseño moderno tipo dashboard (dark UI, responsive)
JavaScript (Vanilla); lógica, simulación y estado global
Chart.js; visualización de datos en tiempo real
Groq API (Llama 3.3 70B); análisis inteligente de decisiones
🔥 Funcionalidades clave

Dashboard ejecutivo en tiempo real; simulación financiera completa; sistema de decisiones estratégicas; asesor IA integrado; eventos aleatorios de mercado; sistema de empleados dinámico; visualización de métricas empresariales; chat interactivo con IA.

📊 Métricas simuladas

Capital disponible; ingresos mensuales (MRR); gastos operativos; burn rate; runway; moral del equipo; rentabilidad; riesgo operativo.

🤖 Inteligencia Artificial aplicada

Asesor empresarial basado en Llama 3.3 70B; análisis contextual en tiempo real; generación de recomendaciones estratégicas; cálculo automático de impacto (delta financiero y operativo); simulación de escenarios empresariales.

💡 Valor diferencial

Proyecto orientado a negocio y no solo a código; combina desarrollo + análisis + simulación; demuestra cómo aplicar IA en decisiones reales; interfaz similar a productos SaaS modernos; enfoque en métricas reales usadas en startups.

🧩 Arquitectura

Frontend puro sin frameworks; manejo de estado centralizado; renderizado dinámico del DOM; integración directa con API externa; lógica de simulación desacoplada; sistema escalable para futuras mejoras.

🚀 Cómo ejecutar
Clonar el repositorio
Abrir el archivo index.html
Usar el dashboard y tomar decisiones
Interactuar con el asesor IA
Analizar resultados en tiempo real
⚠️ Consideraciones técnicas

La API key está expuesta en el frontend solo con fines demostrativos; en producción se debe usar backend y variables de entorno; el sistema es una simulación, no usa datos reales; diseñado para demostrar lógica y arquitectura.

📈 Posibles mejoras

Backend con persistencia; autenticación de usuarios; multiusuario (simulación competitiva); conexión con APIs reales; sistema de guardado de progreso; migración a React o arquitectura modular.
